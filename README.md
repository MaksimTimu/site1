Название проекта: "Ручная работа"

Описание
Одностраничный сайт для мастера ручной работы Ирины Тимуш. Сайт включает разделы: главная, «Обо мне», «Мои работы», «Техники и материалы», «Заказать изделие», «Контакты».

Функциональные возможности
Адаптивный дизайн для всех устройств
Динамическая навигация без перезагрузки страниц
Мобильное меню с анимацией
Форма заказа с клиентской валидацией
Интерактивная карта Yandex в разделе контактов

Технологии
Frontend: HTML5, CSS3, JavaScript (ES6+)
Стилизация: CSS Grid, Flexbox, CSS-переменные, медиа-запросы
Оптимизация: минифицированные CSS/JS, сжатые изображения
